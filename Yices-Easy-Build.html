Description

Provides a monad transformer for constructing Yices queries.

This is not an essential part of the interface. Expressions may be built using Yices.Easy.Types directly.

The monad is simply a convenient way to handle bookkeeping regarding fresh names, accumulated declarations / assertions, and choosing variables to retrieve from the Model.
