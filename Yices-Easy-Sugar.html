Description

Syntactic sugar for embedding Yices expressions in Haskell.

Defines a number of aliases and infix operators, and a Num instance for Expr.

This is not an essential part of the interface. Expressions may be built using Yices.Easy.Types directly.

This is a quick hack. There are several other projects working on vastly more sophisticated SMT EDSLs, so this one is unlikely to receive much attention. That said, suggestions and patches are always welcome.
